# Abbado Website

Documentation website for [Abbado](https://github.com/abbado-dev/abbado), built with [Astro](https://astro.build) + [Starlight](https://starlight.astro.build).

Live at [abbado.dev](https://abbado.dev).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deployment

Deployed to Cloudflare Pages. Pushes to `main` trigger automatic builds.
