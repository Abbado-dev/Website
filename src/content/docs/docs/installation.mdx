## Quick install

```bash
curl -fsSL https://abbado.dev/install.sh | sh
```

This downloads the latest pre-built binary for your platform (macOS arm64/x86, Linux x86) and installs it to `/usr/local/bin/`.

Then install at least one AI CLI agent:

```bash
npm install -g @anthropic-ai/claude-code
claude login
```

Start the dashboard:

```bash
abbado
```

Open [http://localhost:3000](http://localhost:3000).

<Aside type="note">
No Docker required. No Rust or Node.js needed. The binary includes everything.
</Aside>

## Optional tools

| Tool | Purpose | Install |
|------|---------|---------|
| **Codex CLI** | OpenAI agent support | `npm install -g @openai/codex` |
| **GitHub CLI** | PR creation for GitHub repos | `brew install gh && gh auth login` |
| **GitLab CLI** | MR creation for GitLab repos | `brew install glab && glab auth login` |

## Build from source

Requires Rust 1.92+ and Node.js 22+.

```bash
git clone https://github.com/abbado-dev/abbado.git
cd abbado
make build
./target/release/abbado
```

For development with hot reload:

<Tabs>
  <TabItem label="Dev mode">
    ```bash
    make dev    # Backend + frontend in parallel
    ```
  </TabItem>
  <TabItem label="Manual">
    ```bash
    cargo run &
    cd frontend && npm install && npm run dev
    ```
  </TabItem>
</Tabs>

In dev mode, the frontend runs on `http://localhost:5173` with API proxy to the backend on port 3000.

## Environment variables

| Variable | Default | Description |
|----------|---------|-------------|
| `ABBADO_DATA_DIR` | `./data` | Directory for SQLite database and worktrees |
| `ABBADO_BIND` | `127.0.0.1:3000` | Server bind address |
| `ABBADO_FRONTEND_DIR` | *(embedded)* | Override frontend assets path (dev only) |
| `RUST_LOG` | `abbado=debug` | Log level |

## Verify

```bash
abbado --version
```

After starting Abbado, open [http://localhost:3000](http://localhost:3000). You should see an empty dashboard. Click **"+ New Project"** to get started.
